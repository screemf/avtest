#####Автотесты на приложение BLOG######
blogapp - Это приложение написанное в качестве задания по курсу Python Developer. Basic - так что оно не лишено ряда дефектов
часть функций плохо поддается автотестированию (Функционал связанный с "Авторством" - 1 пользователь может однократно отправить
заявку - что в итоге рождает зависимый тест на отправку заявки на автороство - фактически требуется - регистрация
- апйдет профиля - отправка заявки на авторство), часть функций внутри приложения работает "не совсем приятно"- подробнее будет
описано в допущенях.
Для проведения тестирования нам понадобятся 3 группы пользователей
1. Сотрудник - Автор
2. Не сотрудник - Автор
3. Не сотрудник - Не Автор (Можно также добавить Сотрудник - но не Автор)
Instalation
Python 3.10.12
Cписок пакетов в файле requirements.txt
Как запустить:
Собираем локальный образ этого приложения в контейнер "my-app-fr"
В Jenkins - указываем в найстроках pipline script from SCM = Git выбираем в качестве репозитроия основное прложение
